# chatGpt_API

For launch backend :
- ````cd server````
- ````npm run server````

For launch frontend :
- ````cd client````
- ````npm run dev````

# Deployment : 
This is a project using openAI (https://openai.com/api/) 

- The backend hosting on render : https://dashboard.render.com/
- The application hosting on vercel : https://chat-gpt-api-seven.vercel.app/
